# laserShotSynth

![image](https://user-images.githubusercontent.com/66480309/145033683-4d43c9d6-3743-492f-bb12-2bded90a9c97.png)

## Features:
	•	Polyphonic laser shot synthesizer
	•	Sine sweep oscillator
	•	Tune knob
	•	Frequency envelope knob
	•	Decay time knob
	•	Trigger button
	•	Midi note handling

## Installing instructions (Mac OSX):

	•	Download: https://drive.google.com/drive/folders/12Zx3sl15iePl8FsUURne8hUMbQIgs3h1?usp=sharing
	•	Standalone app: Execute the laserShotSynth.app
	•	VST3: Copy laserShotSynth.vst3 into your VST3 folder and use it in your DAW
	•	Component: laserShotSynth.component into your components folder and use it in your DAW
